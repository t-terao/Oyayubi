# Oyayubi

メガネにセンサをつけてモーターを連動させ顔の動きに合わせてカメラを回転させることを目標とする

# ファイル構造
 ・blootooth接続確認用------blootooth接続に必須であるmacアドレス確認とサーバー側の入力をクライアント側に送るサンプルプログラム
 ・Wi-fi接続確認用----------Wi-fi接続に必須であるmacアドレス確認用ソースコード
 ・サーバー側ソースコード---実動作の際のサーバー側ソースコード、I2C接続用とblootooth用
 ・モーター側ソースコード---実動作の際のモーター側ソースコード、I2C接続用とblootooth用
 ・初期動作確認-------------センサー類の初期動作確認用に使ったソースコード
 ・データシートなど---------ESP32のピン配置図と加速度センサのソースコード、実際に動作させた際の動画
